# 选项

提供基础布局，样式自定义。

# 组件概述

  • l-layout ：布局容器，其下可嵌套 Header Sider Content Footer 或 Layout 本身，可以放在任何父容器中。

  • l-header ：顶部布局，其下只可嵌套 Layout，只能放在 Layout 中。

  • l-sider ：侧边栏，其下只可嵌套 Layout，只能放在 Layout 中。

  • l-content ：内容部分，其下只可嵌套 Layout，只能放在 Layout 中。

  • l-footer ：底部布局，其下只可嵌套 Layout，只能放在 Layout 中。
